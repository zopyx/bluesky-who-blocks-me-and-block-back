Features · Built for Power

Everything you need
to moderate at scale.

Purpose-built for Bluesky moderators — manage accounts, lists, rules, and audits from a single native workspace.

👤

Multi-Account

Add unlimited accounts with handle + app password, including custom PDS servers. Multi-tenant by design — tag accounts with labels, sync via iCloud, and enjoy automatic session recovery.

📋

List Management

Create lists with pre-built templates, import via CSV/text with preview, and export as CSV/JSON/XLSX/ODS.

🔍

Profile Inspection

Full profile views with relationship badges, post and media browsers, handle history, and list memberships.

⚡

Automation & Rules

Save reusable action presets and create if-then moderation rules. Queue up to 5 concurrent actions with automatic retry.

📊

Audit & Analytics

Track every action in the operation log, view dashboard charts by operation type, analyze engagement metrics, and generate plain-text moderation reports.

🔒

Security & Privacy

Face ID / Touch ID lock with configurable auto-lock. No tracking, no ads, no data collection.

How It Works · Three Steps

From setup to action
in seconds.

Designed for speed. Add and organize accounts, inspect profiles and lists, then automate actions — all without leaving the app.

1

Add & Organize

Sign in with any Bluesky handle + app password. Add unlimited accounts, tag them with labels, and sync metadata across devices via iCloud.

2

Inspect & Analyze

Browse profiles with full relationship context, posts, media, and handle history. Bulk-lookup accounts, compare lists, and track membership growth.

3

Automate & Audit

Create action presets and moderation rules. Queue actions with parallel execution. Track everything in the operation log and generate reports.

No Tracking

Completely Free

Every feature ships with the app. No upsells, no premium unlocks, no feature gating.

Open Source

Keychain Storage
